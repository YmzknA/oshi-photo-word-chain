# 推し photo
*～好きなものの写真でしりとりをするアプリ～*  
https://oshiphotowordchain-xbcvef1t.b4a.run/

## サービス概要  
  好きなものの写真でしりとりをするアプリです。写真に映る好きなものについて、調べたりすることで、より深く知って貰えます。

## このサービスへの思い・作りたい理由 
  自分の好きなものを共有するときの口実が欲しくて作りました。しりとりの要素を加えることで、遊び心を交えながら交流できます。

## ユーザー層について  

- **好きなものの写真を共有したい人**  

- **皆の好きなものが観たい人**
